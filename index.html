PROTOCOLE D'INTRUSION

En cas d'intrusion hostile (humaine ou zombie), veuillez suivre les étapes suivantes:

  1. Alerter tous les membres présents via le système d'alarme
  2. Évacuer les non-combattants vers le bunker de sécurité

  4. Utiliser uniquement des armes silencieuses si possible
